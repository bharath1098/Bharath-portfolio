# Marriage Portfolio Website

A premium, animated marriage biodata portfolio site built with React, Tailwind CSS, and Framer Motion.

## Location

`E:\marriage-portfolio`

## Features

- **3 languages**: English, Kannada, Hindi
- **Animated sections**: scroll reveals, hero parallax, skill bars, lightbox gallery
- **Sections**: Hero, About, Personal Details, Career, Family, Preferences, Gallery, Contact
- **WhatsApp** floating button
- **Editable content**: `src/data/profile.json` + `public/locales/*.json`

## Quick Start

```bash
cd E:\marriage-portfolio
npm install
npm run dev
```

Open http://localhost:5173

## Customize

1. Edit **`src/data/profile.json`** — name, age, phone, email, photos, skills
2. Edit **`public/locales/en.json`**, **`kn.json`**, **`hi.json`** — all translated text
3. Replace photo URLs with your own images

## Build for production

```bash
npm run build
npm run preview
```
